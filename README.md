<h2>Main Features ⚡</h2>
<ul>
  <li>Insira anotações no bloco de notas</li>
  <li>Edite e exclua quantas anotações quiser</li>
  <li>Mesmo após fechar a página, as anotações ficam salvas no LocalStorage</li>
</ul>
